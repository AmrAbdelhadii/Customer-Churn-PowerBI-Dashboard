# 📊 Customer Churn Analysis - Power BI

## 📌 Project Overview
This project analyzes customer churn for a telecom company using Power BI to identify key factors affecting customer retention.

## 🎯 Objectives
- Analyze churn rate and customer behavior
- Identify key drivers behind churn
- Provide actionable business recommendations

## 📊 Dashboard Highlights
- Churn Rate Overview
- Churn by Contract Type
- Payment Behavior Analysis
- Customer Segmentation (Age Groups)
- Key Insights & Recommendations

## 💡 Key Insights
- Customers with month-to-month contracts have higher churn rates
- Certain payment methods are associated with increased churn
- Younger customer segments show different churn patterns

## 🛠️ Tools Used
- Power BI
- DAX
- Data Visualization
- Power Query
